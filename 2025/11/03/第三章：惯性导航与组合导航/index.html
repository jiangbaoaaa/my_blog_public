第三章：惯性导航与组合导航

发表于 2025-11-03 更新于 2026-07-08 分类于 自动驾驶与机器人中的SLAM技术

自动驾驶与机器人中的SLAM技术——第三章：惯性导航与组合导航

IMU系统的运动学

激光SLAM

问题：

  1. 频率低
  2. 运动畸变
  3. 特征稀疏场景失效

惯性导航

IMU通常包含加速度计和陀螺仪，可以高频测量设备自身的三轴加速度和三轴角速度。

作用：

  1. 高频位姿估计
  2. 运动畸变校正
  3. 为激光匹配提供初始值

缺点：

IMU是通过对加速度和角速度进行积分来推算位姿的，这个过程存在累计误差。时间一长，哪怕是很小的噪声也会被不断放大，导致推算出的位置和姿态会像“脱缰的野马”一样飘走（即漂移），无法长期独立工作。

组合导航

组合导航就是将激光SLAM和惯性导航组合起来，形成一个激光惯性SLAM系统。核心思想：

用IMU高频、短时精确的特性，弥补激光雷达的低频和畸变问题；同时，用激光SLAM长期、全局精确的特性，来不断校正IMU的累计误差。

技术上主要分为两种融合方式：

  1. 松耦合：这是一种比较简单直接的融合方式。激光SLAM和IMU各自独立计算出自己的位姿，然后通过一个滤波器（如卡尔曼滤波器）将这两个“结果”进行融合。它实现简单，但因为没有利用原始数据，精度和鲁棒性相对较差。
  2. 紧耦合：这是一种更先进、更主流的融合方式。它将IMU的测量值（角速度、加速度）和激光雷达的原始特征（点、线、面）共同放入一个大的优化问题中进行求解（通常使用因子图优化或复杂的卡尔曼滤波器）。在这种框架下，激光的观测可以非常直接地校正IMU的状态（如零偏），IMU的数据也可以更精确地指导点云处理。紧耦合虽然复杂，但性能更优，鲁棒性更强，是当前研究和应用的热点。

这样就得到了纯粹的运动加速度，如果对速度进行一次时间积分，就得到了速度，对速度进行再次时间积分，就得到了位置。

需要注意的是，要把IMU放在车辆中心，否则可能会在车辆发生旋转和移动的时候，测量到由车辆旋转导致的离心力、科氏力和角加速度。会影响IMU的读数。

IMU测量方程中的噪声模型

IMU噪声由两部分组成：测量噪声和零偏。在实际卡尔曼滤波算法中，系统状态不仅包括姿态、位置、速度，还会把偏置作为一个状态量来进行在线估计和校正，抑制IMU的漂移。噪声是滤波器的重要输入，通常通过艾伦方差工具对IMU进行离线标定获得。

使用IMU进行航迹推算

在已知此时刻位置速度以及姿态的前提下，通过测量IMU数据，获得角速度以及加速度，来更新姿态以及位置速度。

航迹推算是一个迭代积分的过程，在每个时间步，先用陀螺仪积分更新姿态，再利用新姿态消除加速度计读数中的重力，最后对纯运动加速度进行二次积分来更新速度和位置。在实际系统中，必须与其他传感器（GPS、LiDAR、相机等）进行融合，由这些外部传感器来不断校正IMU的累积误差，从而实现长期、鲁棒的定位。

卫星导航

我们日常生活中最熟悉的“GPS”其实是美国开发的系统的名称，而这类系统的通用术语是GNSS（Global Navigation Satellite System，全球卫星导航系统）。卫星导航实际上可以提供车辆所需的所有定位信息，包括车辆的位置、姿态、速度等物理量。

卫星定位的最终精度受很多因素影响，人们提出各种校正技术消除已知误差。PPP（Precise Point Positioning 精密单点定位）、**RTK(Real-Time Kinematic 实时动态差分)**等技术取得进展，使得定位精度不断提升。

差分全球卫星导航系统中差分技术是消除误差最核心，最有效的方法之一，基本思想是“通过比较消除共同误差”

主要技术

  • SBAS (星基增强系统)：这是一种广域差分系统。它由多个地面站组成网络，计算出大范围（如整个大陆）的修正信息，然后通过地球同步卫星广播给用户。例如美国的 WAAS、欧洲的 EGNOS，以及覆盖日本的 MSAS。
  • RTK (实时动态) 技术：这是一种更高精度的载波相位差分技术。它不仅使用差分修正伪距，还利用了信号的载波相位进行测量。通过解算整周模糊度，RTK技术可以实现厘米级的实时定位精度，是测绘、精准农业和自动驾驶等领域的核心技术。

两个蘑菇头就是RTK接收器，通常使用两个接收器组成双天线方案，根据两根天线的位置差计算车辆实时朝向。

常见的世界坐标系

地球上最常见的坐标系就是经纬度坐标系，即地理坐标系。而UTM（Universsal Transverse Mercator Grid System）坐标系是将地球视为一个椭球体，投影至横躺圆柱体上，将其展开并且进行分区得到的。

使用UTM的优点是使用了米制单位，与其他传感器兼容性好，缺点是某些地区可能存在两个分区的跨界处，需要进行额外的坐标处理。

使用误差状态卡尔曼滤波器实现组合导航

RTK设备为我们提供一个不太稳定的位姿观测源，可以使用RTK与IMU结合，使用拓展卡尔曼滤波器形成组合导航算法。严格来说，是使用误差状态卡尔曼滤波器（Error-State Kalman Filter, ESKF）。

误差状态卡尔曼滤波器的核心思想是：不直接估计整个状态，而是将状态分为两部分：一个大的“标称状态”和一个小的“误差状态”。

  1. 标称状态 (x): 使用IMU的原始测量值，通过非线性运动学方程进行高频积分。这个状态包含我们主要关心的量：位置 p、速度 v、姿态 q（通常用四元数表示）。
  2. 误差状态 (δx): 使用一个线性的卡尔曼滤波器来估计标称状态中存在的微小误差。这个误差状态通常包括：位置误差 δp、速度误差 δv、姿态误差 δθ（一个小的旋转矢量）、以及IMU的偏置误差 ba,bg.

整个过程是一个经典的“预测-校正”循环。

速度观测量

在GINS（GNSS/INS）系统中，长时间缺少RTK数据，ESKF就变为纯靠IMU积分的递推模式。该模式下位移将快速发散， 因为缺少速度观测。大部分轮式机器人会携带一个编码器测定自身速度，进而推算自身的局部运动情况。这些速度观测与IMU结合，可以形成局部的航迹推算，可以简单将他们融入ESKF，作为速度观测量。

轮式编码器它被安装在车轮的转轴或电机上，用于测量车轮的旋转运动。通过测量轮子转了多少圈、转了多快以及朝哪个方向转，我们就可以间接计算出机器人或车辆的行驶距离、速度和方向。

其特点可以概括为：

  1. 单个轮式编码器只输出轮子转过的距离，可以对这个距离进行采样。如果每隔固定时间采样，就可以看成对速度的测量值。
  2. 如果本身在底盘上的角度是固定不动的，可以通过编码器计算车辆位移
  3. 轮子本身智能测量车辆在前进方向上的速度位移，无法测量平行方向上或者高度的速度位移
  4. 实际中，车轮容易打滑，轮子可能会发生空转，导致编码器虽然有输出，胆识车辆本身并没有前进的状态。

在代码中，可以添加其X方向上的速度作为观测模型，可以获得带有轮速的ESKF。
