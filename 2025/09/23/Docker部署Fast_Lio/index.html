创建并运行FAST_LIO的Docker容器

  1. 创建执行脚本

主目录下新建文件

脚本内容理解

  • mkdir docker_ws: 这会在当前目录下创建一个名为 docker_ws 的文件夹。这个文件夹将作为您的主机和Docker容器之间的共享文件夹。之后下载的数据集就要放在这里。

  • xhost +local:: 这条命令允许容器内的图形界面程序（比如RViz可视化工具）显示在您的主机桌面上。

  • docker run -itd …: 这是核心命令。

  • 它会从Docker Hub上下载 kenny0407/marslab_fastlio2:latest 这个镜像文件。这个镜像已经包含了Ubuntu, ROS, FAST-LIO以及所有必需的库。下载速度取决于您的网络情况，请耐心等待。

  • –name=”fastlio2”: 给创建的容器命名为 fastlio2。

  • -v /home/$USER/docker_ws:/home/mars_ugv/docker_ws: 这是最关键的一步，它将您主机上的 docker_ws 文件夹映射到容器内的 /home/mars_ugv/docker_ws 文件夹。这意味着，您在主机 docker_ws 文件夹里放任何文件，容器里会同步出现，反之亦然。
