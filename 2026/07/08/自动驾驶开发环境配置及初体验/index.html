仿真运动

给定目标点

  1. 选择工具： 将鼠标移到 RViz 界面最上方的工具栏，找到并点击 2D Goal Pose 按钮（图标通常是一个带箭头的标记）。

  2. 在地图上画出终点

    • 将鼠标移动到右侧黑白点云地图上的一条有效车道上（最好在车辆前方一段距离）。

    • 按住鼠标左键不放，顺着你期望的停车方向（车道前进方向）轻轻拖动一下鼠标，然后松开。

  3. 验证现象：

    • 地图上会立刻生成一条从当前位置延伸到终点的彩色渐变轨迹线。

    • 左侧的 STATE 会从 WAITING_FOR_ROUTE 变成亮蓝色的 WAITING_FOR_ENGAGE（等待激活）。

点击Engage 发车

路线规划出来后，车还处于挂着 P 挡（PARKING）拉着手刹的状态。

  1. 寻找 Engage 按钮： 在左侧 AutowareStatePanel 面板中，往下看，找到写着 Engage: False 字样上方或旁边对应的 Engage 按钮（有时它就是一个可以直接点击的长条按钮，或者需要展开）。
  2. 点击激活： 用鼠标点击它。
  3. 验证现象：
    • 状态栏的 GEAR 会从 PARKING 切换到 DRIVE。
    • STATE 会变成绿色的 DRIVING（行驶中）。
    • 界面左上角那个白色的方向盘会开始自动转动，旁边的时速表（0.00km/h）数字会逐渐变大，系统坐标系或隐藏的小车就会沿着那条彩色轨迹跑起来了
